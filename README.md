# Taller de Lenguajes II - Trabajo práctico nº 1

## Ejercicio 2 - respuestas

- Los Clientes se relacionan con los Pedidos mediante composición ya que, al eliminarse un Pedido, necesariamente se debe eliminar el objeto Cliente.
  Los Pedidos se relacionan con los Cadetes mediante agregación, puesto que los pedidos pueden ser reasignados a otro Cadete.
  Los Cadetes se relacionan con la Cadetería mediante agregación dado que los cadetes podrían existir independientemente de un objeto Cadeteria.
